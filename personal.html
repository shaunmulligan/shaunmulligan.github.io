Push notification Doorbell Android. RPI. Node.js

To help my sisters guesthouse run smoother, I built a raspberry pi door bell system. The system pushes door notifications using GCM to the employee that is currently on duty. The system also allowed the employees to clock-in and out of work using the app, these times were updated on a google spread sheet automatically. The system was built using node.js and express on the raspberry pi as well as some node modules to access the GPIO of the RPI.The app was built using java on the android platform.

Django Learning Django web framework for Python

To start learning the Django framework, I built a basic webphoto viewer. The idea is to have a webenabled picture frame for my mother and gran. It will allow them to connect to a dropbox folder which contains all their photos. It will also allow them to follow Facebook and Instagram photos from their children and grandchildren. Currently I have only implemented the dropbox enabled photo viewer app. Ultimately when the project is finished I hope to build a small stylish digital frame that runs a broswer in full screen. I also plan to add some kind of hash-tag filter, so that not all my facebook photos are directly streamed to my Granma, because that could be dangerous :P
